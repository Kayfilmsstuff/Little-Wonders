# Tiny Wonders — NFC Verify Site

Upload this folder to a public GitHub repo and enable GitHub Pages (Settings → Pages → Branch: `main`, Folder: `/root`).

- Program your NFC tag with a URL like: `https://<yourname>.github.io/<repo>/verify.html?code=7F9X2B`
- Lock the NTAG215 after testing.
- Add new items by editing `data/items.json` and adding images in `/images`.
